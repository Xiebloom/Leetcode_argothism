## 3 分析：

### 3.1 时间复杂度

* 好 O( n+k )  //每个数据都有一个桶
* 坏 O ( n² )  //只有一个桶

### 3.2 空间

* O ( n+k ) //k个数据n个桶

### 3.3 稳定性

* 标准答案：稳定
* up的理解：与桶内的排序有关，如果桶内是用快速排序这种不稳定的算法的话，自然是不稳定的。
